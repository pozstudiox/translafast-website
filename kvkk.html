KVKK Notice

According to the Turkish Law on the Protection of Personal Data No. 6698 (“KVKK”), the protection of personal data is a fundamental right. As TranslaFast, we respect this right and do not process any personal data. This disclosure has been prepared to fulfill our legal information obligation.

1. About the Data Controller

TranslaFast operates as a web translation extension and does not store any user data on any server or system. As the data controller, TranslaFast does not engage in any personal data processing activities because it does not record any data.

2. Personal Data Processed

TranslaFast does not process any personal data. During use, no identity, contact, location, usage habits, IP address, or similar data is collected. The extension operates solely on the user's device, and all operations are local.

3. Explicit Consent and Data Sharing

There are no data processing activities that require obtaining explicit consent from users. The extension does not include any cookie-based or advertising-based tracking mechanisms. It is not integrated with any third-party analytics, advertising, or sharing services.

4. No Purpose for Personal Data Processing

TranslaFast is solely a translation tool that prioritizes user privacy. No data is collected for marketing, analytics, profiling, or reporting purposes.

5. Data Retention and Security

Since no data is collected, there are no defined retention periods or data security policies. However, the application's code is developed in accordance with security principles. All processes are contained within the user's device and do not leave the application.

6. Your Rights

Under the KVKK, individuals have rights such as access to data, correction, deletion, and objection to processing in cases where data is processed. Since TranslaFast does not process any data, these rights do not directly apply to the extension; however, users may contact us with any questions at pozstudiosh@gmail.com.

7. Updates

This disclosure may be updated periodically for legal compliance or informational purposes, even if no operational changes occur. The latest version will always be published on this page.
