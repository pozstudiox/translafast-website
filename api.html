API Anahtarı

TranslaFast API'yi kullanmak için aşağıdaki API anahtarınızı kullanabilirsiniz.

API Kullanım Rehberi

API'nizi nasıl kullanacağınız hakkında detaylı bilgi edinin.

API Anahtarı Nasıl Alınır?

TranslaFast hesabınıza giriş yaparak ücretsiz bir API anahtarı oluşturabilirsiniz.

API Sınırları

Günde maksimum 10.000 istek yapabilirsiniz.

Geliştirici Belgeleri

Tüm uç noktalar ve yanıt formatları detaylı şekilde belgelenmiştir.

Ücretlendirme

Temel kullanım ücretsizdir. Profesyonel planlar için iletişime geçin.

API Kullanım Örneği

fetch('https://api.translafast.com/translate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer TF-API-9876-1234-ABCD',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ text: 'Merhaba', target_lang: 'en' })
})
.then(response => response.json())
.then(data => console.log(data));

Hızlı Başlangıç

İlk 5 isteğiniz örnek cevaplarla hızlı şekilde test edilebilir.

Yanıt Biçimi

Tüm API cevapları JSON formatındadır.

Güvenlik

API anahtarınızı kimseyle paylaşmayın ve gizli tutun.

API Versiyonları

Şu anda v1.0 sürümü aktiftir. Gelecekte v2.0 desteklenecektir.

İstek Limiti Arttırımı

Limit arttırımı için destek ekibimize başvurabilirsiniz.

Hata Kodları

Her hata kodu ilgili açıklamayla birlikte döner.

Test Ortamı

Test ortamı anahtarlarıyla canlı sisteme zarar vermeden deneyimleyebilirsiniz.

Başarılı İstek Örneği

200 OK yanıtı başarılı bir isteği gösterir.

Yetkilendirme

Her istekte Bearer Token kullanılması zorunludur.

IP Kısıtlaması

İsteğe bağlı olarak belirli IP adreslerine API kullanımı sınırlandırılabilir.

Webhook Desteği

Çeviri tamamlandığında webhook ile otomatik bildirim alabilirsiniz.

Kullanım Koşulları

API kullanımı TranslaFast kullanım koşullarına tabidir.

Sıkça Sorulan Sorular

API hakkında sık sorulan soruları ve cevapları okuyabilirsiniz.

Canlı Destek

Herhangi bir sorunda 7/24 destek alabilirsiniz.

Performans Ölçümleri

İstek yanıt sürelerini ve işlem sürelerini takip edebilirsiniz.

Veri Politikası

Gönderdiğiniz içerikler gizlilikle korunur, üçüncü kişilerle paylaşılmaz.

Versiyon Geçişleri

Yeni sürümlere geçiş süreci hakkında bilgilendirme yapılır.
