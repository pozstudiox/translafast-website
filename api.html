TranslaFast API Dökümantasyonu

TranslaFast API ile kendi uygulama veya sitelerinizde hızlı ve kolay çeviri entegrasyonu sağlayabilirsiniz.

🔑 API Anahtarı

Her istek için bir API anahtısı gerekir. Kayıt olduktan sonra panelden anahtarınızı alabilirsiniz.

X-API-Key: senin-api-anahtarın

📨 Çeviri İsteği Gönderme

POST isteği ile aşağıdaki uç noktaya çeviri isteği gönderin:

https://translafast.onrender.com/api/translate

📦 Gönderilecek Veri (JSON)

{ "text": "Merhaba dünya", "from": "tr", "to": "en" }

📝 Python Örneği Python

import requests url = "https://translafast.onrender.com/api/translate" headers = {"X-API-Key": "senin-api-anahtarın"} data = {"text": "Merhaba", "from": "tr", "to": "en"} res = requests.post(url, json=data, headers=headers) print(res.json())

💻 JavaScript Örneği JS

fetch("https://translafast.onrender.com/api/translate", { method: "POST", headers: { "Content-Type": "application/json", "X-API-Key": "senin-api-anahtarın" }, body: JSON.stringify({ text: "Hello world", from: "en", to: "tr" }) }) .then(res => res.json()) .then(data => console.log(data));
